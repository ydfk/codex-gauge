# Design

Codex Gauge 的界面目标是低干扰、高信息密度和桌面背景上的稳定可读性。

## macOS 菜单栏

- 应用仅常驻右上角菜单栏，不显示 Dock 图标或桌面浮窗
- 直接缩放 Windows 托盘图标，保留深色圆底、薄荷色圆环、白色 G 和橙色提示点，并显示 `5h`、`7d` 剩余比例
- 菜单栏标题使用系统文字色，详情面板的百分比沿用 Windows 状态色和阈值
- 单击展开 360 × 480 详情面板，失焦或按 Esc 收起
- 详情与设置在同一个面板内切换
- 发现新版本时在详情顶部提示，点击后自动安装并重启
- 使用系统字体并适配系统深浅色，双轨额度条是主要识别元素
- 不提供位置、置顶、透明度、锁定或 OLED 位移设置

## 视觉

- 深色玻璃拟态
- 小尺寸圆角浮窗
- 细线、圆环和状态点表达状态
- 不使用大面积鲜艳颜色
- 不使用复杂动画或图表库

## 状态色

- `0%~59%`：正常
- `60%~84%`：注意
- `85%~94%`：警告
- `95%~100%`：危险
- 未登录或 CLI 异常：灰色

## 默认信息

```text
Codex Gauge
5h  42%  02:18
1w  17%  3d 08h
R2  Today 128k
```
